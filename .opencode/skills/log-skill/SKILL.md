## What I Do

- Tambahkan log yang sudah dikerjakan ke dalam file `LOG.md` di repository ini.

## When to Use me

- Setelah selesai membuat perubahan atau menambahkan fitur baru, gunakan saya untuk mencatat apa yang sudah dikerjakan.

## How I do it

1. Read `LOG.md` untuk melihat log terakhir yang sudah dicatat
2. Append entry baru di bagian bawah file `LOG.md`
3. Group berdasarkan tanggal, gunakan bullet points
4. Format : aksi ( `Modified`, `Created`, `Fixed`, `Deleted` ) + nama file atau fitur + deskripsi singkat

## Log Format Example

```markdown
# LOG.md

## 2023-05-12

- Modified `src/feature/new-feature/new-feature.ts` - Update logic untuk handle edge case
- Created `src/feature/new-feature/new-feature.ts` - Implementasi fitur baru untuk user registration
- Fixed `src/feature/bug-fix/bug-fix.ts` - Perbaiki bug pada validasi input form
```

**PENTING:** Selalu append log terbaru di bagian bawah file `LOG.md` untuk menjaga kronologi perubahan. Jangan menghapus atau mengedit log yang sudah ada, kecuali untuk memperbaiki kesalahan penulisan.
